# YumeRobo

A static site for publishing media releases.

## Tech Stack

- **Framework**: [SvelteKit](https://kit.svelte.dev/) with TypeScript
- **Styling**: Vanilla CSS with CSS Custom Properties
- **Build**: [Vite](https://vitejs.dev/)
- **Hosting**: [Cloudflare Pages](https://pages.cloudflare.com/) (Static Adapter)
- **Runtime**: [Bun](https://bun.sh/)

## Credits

- MediaInfo parser adapted from [UNIT3D](https://github.com/HDInnovations/UNIT3D)
